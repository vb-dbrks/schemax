## Python SDK & CLI

```bash
cd packages/python-sdk
pip install -e .
schematic --version
```

### Common commands

```bash
schematic validate                    # Validate project
schematic sql --output migration.sql  # Generate SQL from changelog
schematic snapshot create --name "v0.1.0"
schematic diff --from v0.1.0 --to v0.2.0
schematic apply --target dev --profile DEV --warehouse-id <id> --dry-run
```

## Your first schema (CLI)

```bash
mkdir my-schema && cd my-schema
schematic init   # or open Designer once to create .schematic/
# Add ops via Designer or append_ops; then:
schematic snapshot create --name "Initial" --version v0.1.0
schematic sql --output migration.sql --target dev
```
